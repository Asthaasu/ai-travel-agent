# ✈️ AI Travel Agent

An intelligent AI-powered travel planning assistant that generates complete travel itineraries including flights, hotels, and attractions — all in one place.

🔗 **Live Demo:** [Click Here](https://ai-travel-agent-hof3nkyhtncijbl2jpbp8l.streamlit.app/)

---

## 🌟 Features

- ✈️ Flight search suggestions  
- 🏨 Hotel recommendations  
- 🗺️ Top tourist attractions  
- 🤖 AI-generated travel plans  
- 📧 Email delivery of travel itinerary  
- ⚡ Fast and responsive UI (Streamlit)  
- 💸 100% FREE backend using Groq API  

---

## 🧠 Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **LLM:** Groq (LLaMA 3.1)  
- **APIs Used:**
  - SERPAPI (Google Search)
  - SendGrid (Email service)

## ⚙️ How It Works

1. User enters travel request  
2. System fetches:
   - Flights ✈️  
   - Hotels 🏨  
   - Attractions 🗺️  
3. AI compiles a structured travel plan  
4. Optionally sends plan to email 📧
